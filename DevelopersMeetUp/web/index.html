A programming language is a formal computer language or constructed language designed to communicate instructions to a machine, particularly a computer.
Programming languages can be used to create programs to control the behavior of a machine or to express algorithms.


Fact About Different Programming Languages

C

  • C was created in 1972 by Ken Thompson and Dennis Ritchie, researchers at AT&T's Bell Labs in Murray Hill, New Jersey.
  • C is a high-level programming language that is used to develop many kinds of software.
  • Although it is a high-level language, C is capable of controlling the computer on which it operates at a low level.

Java

  • Sun Microsystems created Java. Oracle now oversees Java after its acquisition of Sun in 2010.
  • Java was originally called Oak.
  • There are 9 million Java developers worldwide.
  • More than 95% of enterprise desktop's run Java
  • After the programming language C, Java ranks No. 2 in popularity among programming languages.
  • Java’s mascot, Duke, was created by Joe Palrang.
Meet Java Developers

PHP

  • PHP originally stood for Personal Home Page.
  • PHP which is now officially known as Hypertext Preprocessor was released in the year 1995.
  • PHP was originally created by Rasmus Lerdorf in 1995, and was written in original Common Gateway Interface (CGI) binaries.
  • Some of the biggest online brands, such as Facebook,and Yahoo! are powered by PHP.
Meet PHP Developers

Python

  • In 1991, Guido Van Rossum made python programming language.
  • Python was named after a popular comedy Monty Python’s Flying Circus.
  • Python Language is one of the easiest programming language to learn.
  • Jython is the java implementation of python programming language
